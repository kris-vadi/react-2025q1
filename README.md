## Getting started:

Clone the `react-2025q1` repository using git:

```
git clone git@github.com:kris-vadi/react-2025q1.git
cd react-2025q1
```

Install dependencies with this command:

```
npm install
```

Run the application with this command:

```
npm run dev
```
